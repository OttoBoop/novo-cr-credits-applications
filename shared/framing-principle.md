# Princípio de afiliação

A regra que rege toda aplicação a programas de crédito ou aceleração para o Novo CR.

## Quem aplica

**Otavio Bopp.** Cidadão privado. Alumni EPGE/FGV (graduação em economia, formado em 2025).

## O que é credencial, o que é afiliação

| Item | Categoria | Função em aplicações |
|---|---|---|
| Alumni EPGE/FGV | Afiliação acadêmica institucional | Estabelece relação contínua com instituição de pesquisa |
| Cargo na SUBDEIS (Subsecretaria de Desenvolvimento Econômico, Inovação e Simplificação, Prefeitura do Rio) | Credencial | Prova de execução institucional, acesso à rede municipal |
| Passagem pelo gabinete do vereador Flávio Valle (aliado do Eduardo Paes) | Credencial | Rede política, ângulo de implantação em política pública |
| Apoio de Valdemar Pinho Neto (coord. graduação EPGE, orientador TCC) | Afiliação acadêmica via signatário | Signatário primário para programas de pesquisa |
| Apoio de Rubens Cysne (diretor EPGE) | Afiliação acadêmica via signatário | Signatário alternativo para programas de pesquisa |
| Apoio de Moacyr (prof. EMAp) | Afiliação acadêmica via signatário | Dados; ponte para CTAE |
| Apoio de Mario Andrade (coord. pedagógico CTAE) | Caminho de implantação | Implantação em escola FGV (2027) |

**Credencial ≠ afiliação.** SUBDEIS e gabinete Valle são EVIDÊNCIA de execução, NÃO afiliação institucional. Não são quem assina pelo Otavio, não são quem se beneficia diretamente dos créditos. São o que prova que Otavio sabe trabalhar dentro de instituições, e que abrem caminhos de implantação.

## O que NÃO fazer

- Não escrever "Otavio, funcionário da Prefeitura do Rio" como se fosse afiliação institucional do projeto.
- Não usar SUBDEIS como aplicante (a SUBDEIS não é dona do projeto).
- Não usar FGV como aplicante (FGV é parceria de pesquisa via professores, não dona).
- Não usar Otavio sozinho em programas que exigem professor signatário (CNPq, Anthropic AI for Science, OpenAI Researcher Access, Google Research Credits).
- Não confundir "tenho contatos na FGV" com "sou pesquisador afiliado à FGV". O ponto formal é o professor que assina a carta de apoio.

## Implicação prática

Cada aplicação a programa de pesquisa precisa:
1. **Otavio como pesquisador principal** (com afiliação alumni FGV)
2. **Carta de apoio de Valdemar** (ou alternativo) atestando a afiliação ao projeto de pesquisa
3. **Descrição do projeto** centrada no mérito científico do Novo CR (não no produto comercial)
4. **Credenciais executivas** (SUBDEIS, gabinete Valle) como evidência de capacidade institucional, não como afiliação
