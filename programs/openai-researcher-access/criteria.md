# OpenAI Researcher Access Program — Critérios do programa

Fonte: https://openai.com/form/researcher-access-program/ (página retornou 403 no fetch direto — info abaixo via WebSearch e FAQ pública)
Última verificação: 2026-05-23 via WebSearch.

## Resumo

Programa da OpenAI que subsidia pesquisadores acadêmicos com créditos de API para pesquisa.

## Critérios em verbatim (fontes públicas)

> "active affiliation to an academic institution or other research organization"

> Subsidizes researchers "with limited financial and institutional resources"

> Aplicação via formulário; revisão trimestral.

## O que oferece

- Até **$1.000** em créditos de API OpenAI
- Validade de **12 meses**
- Créditos não-transferíveis

> Observação: valor baixo comparado ao Anthropic AI for Science. Funciona mais como credenciamento e validação acadêmica do que como financiamento real.

## Critérios de avaliação

- Mérito do projeto de pesquisa
- Afiliação acadêmica ativa
- Recursos financeiros/institucionais limitados (priorizado)
- Uso responsável + Usage Policy

## Como aplicar

- Formulário em https://openai.com/form/researcher-access-program/

## Ciclo

- **Revisão trimestral:** março, junho, setembro, dezembro
- Turnaround de **4–6 semanas** após o fim da janela

## Encaixe do Novo CR

**Afiliação acadêmica:** atendida via Otavio (alumni EPGE) + Valdemar como signatário.

**"Recursos limitados":** verdadeiro — projeto pessoal, sem funding institucional, custos de API saem do bolso do Otavio.
