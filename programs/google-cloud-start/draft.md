### Company description (curta)

Novo CR is an early-stage Brazilian EdTech company developing AI-assisted longitudinal data infrastructure for educational assessment. The product, in alpha at https://ia-educacao-v2.onrender.com, runs a six-stage AI pipeline that turns each student exam into a narrative report, with the teacher keeping the final decision at every stage. The system processes 26 subjects and over 100 activities across multiple model providers (Anthropic, OpenAI, Google).

### How the credits will be used

Google Cloud credits will support Gemini API usage for production-scale evaluation across the pipeline's six stages, plus exploration of Vertex AI for the cross-course longitudinal report. Specifically:
- Per-stage benchmarking of Gemini against alternatives
- Refinement of the batch pipeline for cross-course longitudinal reports
- Pilot deployment infrastructure for FGV K–12 school in 2027

### What we'd ask for

The standard Start tier benefit: $2,000 GCP credits + $200 training credits for 12 months, used primarily for Gemini API and Vertex AI evaluation.

### Why we'd apply

The credits would reduce friction at a stage where the project transitions from research prototype to commercial pilot. Coupled with FGV Ventures batch 20 acceleration (in evaluation), Google Cloud Start tier would directly support the FGV pilot school deployment planned for 2027.
