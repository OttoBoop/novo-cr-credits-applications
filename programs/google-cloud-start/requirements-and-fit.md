## 2. Decisão estratégica

Para aplicar a este programa, Otavio precisaria **incorporar uma empresa**. Esta é uma decisão maior do que o ganho de $2.200 em créditos justifica isoladamente.

**Cenário "incorporar":**
- MEI: barato (~R$0/ano de registro, ~R$70/mês de DAS), mas tem limites de receita e atividade que podem não casar bem com EdTech B2B.
- Ltda: ~R$2.000-5.000 iniciais com contabilidade, custo mensal de contabilidade ~R$500-1.500. Mais flexível mas operação maior.

**Cenário "não incorporar":**
- Manter como projeto pessoal de pesquisa + alumni FGV.
- Não aplicar aos tracks de startup do Google/OpenAI.
- Focar nos tracks de pesquisa (Anthropic AI for Science, OpenAI Researcher Access, Google Cloud Research Credits).

## 3. Coupling com outras decisões

Esta decisão de incorporar/não-incorporar deve ser tomada em coordenação com:

- **FGV Ventures batch 20:** se Otavio entrar na aceleradora, FGV Ventures pode requerer incorporação ou facilitar o processo (Gust ecosystem).
- **OpenAI for Startups:** mesmo gate — só faz sentido se Otavio for incorporar de qualquer jeito.
- **Google Cloud Scale tier ($200K) ou AI tier ($350K):** os tiers maiores requerem VC funding além de incorporação. Otavio não tem; portanto, mesmo incorporando agora, esses tiers ficam fora.

## 4. Recomendação final

**Adiar a aplicação Google Cloud Start até depois da resposta da FGV Ventures.** Se entrar no batch 20, incorporação vira mais natural — aí aplicar simultaneamente a (a) Google Cloud Start, (b) OpenAI for Startups, e considerar (c) AWS Activate e (d) Microsoft for Startups Founders Hub se valer a pena.

Se Otavio decidir incorporar antes (por outras razões), aplicar Google Cloud Start é trivial — formulário simples, valor baixo, baixo risco.
