# Google Cloud Start (tier) — Critérios do programa

Fonte: https://cloud.google.com/startup/benefits
Última verificação: 2026-05-23 via WebSearch.

## Resumo

Tier inicial do Google for Startups Cloud Program. Para startups pré-funded com MVP. Sem requisito de funding VC.

## Critérios em verbatim (fontes públicas)

> "Built for startups under 5 years old that haven't taken institutional VC funding and have minimal prior Google Cloud credits."

Elegibilidade geral:
- Startup formalmente registrada
- Menos de 5 anos
- Sem funding VC institucional
- Site funcional (verificação de legitimidade)
- Pouco/nenhum uso prévio de créditos Google Cloud
- Associação com aceleradora/incubadora/VC parceiro do Google (para Scale tier — verificar se vale para Start)

## O que oferece

- **$2.000** em créditos Google Cloud (12 meses)
- **$200** em training credits

## Restrição crítica

> "Google does not accept educational institutions, government entities, nonprofits"

Aplica-se aos tiers do programa de startups. Pra Start tier, posicionamento precisa ser **startup**, não projeto acadêmico nem governamental.
